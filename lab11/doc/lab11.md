## ОПИС ПРОГРАМИ

### Функціональне призначення

Програма складається з двох функцій get_diagonal та sort_array і функції main, яка викликає ці функції.

Функція get_diagonal приймає матрицю matrix, розмірність n та вказівник на масив diagonal. Вона записує елементи головної діагоналі матриці n * n у масив diagonal.

Функція sort_array приймає масив array довжиною n та сортує його за зростанням.

У функції main створюється матриця matrix розмірністю N * N, яка містить числа від 1 до 9. Функція get_diagonal записує елементи головної діагоналі цієї матриці у масив diagonal, а функція sort_array сортує елементи масиву diagonal. Нарешті, відсортовані елементи виводяться на екран.

## ВАРІАНТИ ВИКОРИСТАННЯ

Ця програма призначена для знаходження головної діагоналі квадратної матриці та сортування її елементів за зростанням. Програму можна використовувати в більш загальному випадку для знаходження та сортування діагональних елементів квадратних матриць в будь-яких обчислювальних задачах

Першим кроком є введення матриці розміром N x N, після цього програма викликає функцію get_diagonal, яка записує елементи головної діагоналі матриці у одновимірний масив. Потім програма викликає функцію sort_array, яка сортує елементи одновимірного масиву за зростанням. І потів в кінці программа видає відсортовані елементи головної діагоналі. Варто зазначити що елементи повинні бути цілими числами.
